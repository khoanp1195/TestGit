10 Lệnh Git cơ bản

  • 1. Git Init Create a local Repo in your own Pc => git init Link the local repository to an empty GitHub repository => git remote add origin [GitHub_Repo_URL]
  • 2. Git Clone Clone (tải) a Remote Repo on GitHub to Local PC => git clone [GitHub_Repo_REPO]
  • 3. Git Add Add Changes to Staging Area => git add file
  • 4. Git Commit Sbapshots (Ảnh chụp) the file in preparation for versioning => git commnit -m "descriptive message" Để thay đổi đoạn mô tả => git commit --amend -m "amend descriptive message"
  • 5. Git Push Uploads all local branch commits to GitHub => git push origin [branch-name]
  • 6.Git Status to see the status of working directory and staging area => git status
